Introduction

En mathématiques, un système d'équations à deux inconnues est un ensemble de deux équations comportant deux variables (généralement notées x et y). Résoudre ce système consiste à trouver les valeurs des inconnues qui satisfont simultanément toutes les équations du système.

Dans ce cours, nous allons explorer différentes méthodes pour résoudre ces systèmes, comprendre leur signification géométrique et voir diverses applications concrètes.

Représentation graphique d'un système d'équations à deux inconnues : le point d'intersection correspond à la solution.

Qu'est-ce qu'un système d'équations à deux inconnues ?

Un système d'équations linéaires à deux inconnues se présente généralement sous la forme :

{
ax + by = c
dx + ey = f
}

Où a, b, c, d, e et f sont des constantes réelles et x et y sont les inconnues que nous cherchons à déterminer.

La résolution d'un tel système consiste à trouver toutes les paires de valeurs (x, y) qui vérifient simultanément les deux équations.

Interprétation géométrique

Dans un repère cartésien :

  • Chaque équation linéaire représente une droite dans le plan.
  • Résoudre le système revient à trouver les points d'intersection de ces droites.

Trois situations sont possibles :

  • Une solution unique : les droites se coupent en un seul point.
  • Une infinité de solutions : les droites sont confondues (système compatible indéterminé).
  • Aucune solution : les droites sont parallèles sans jamais se couper (système incompatible).

De gauche à droite : système avec une solution unique, une infinité de solutions, aucune solution

Méthodes de résolution

1. Méthode par substitution

Cette méthode consiste à :

  1. Isoler une inconnue dans l'une des équations.
  2. Substituer cette expression dans l'autre équation.
  3. Résoudre l'équation à une inconnue obtenue.
  4. Calculer la valeur de la seconde inconnue en remplaçant dans l'expression isolée à l'étape 1.

Exemple :

Résoudre le système suivant :

{
2x + y = 5   (1)
x - y = 1   (2)
}

Étape 1 : Isolons y dans l'équation (1) :

y = 5 - 2x

Étape 2 : Substituons cette expression dans l'équation (2) :

x - (5 - 2x) = 1

Étape 3 : Résolvons cette équation à une inconnue :

x - 5 + 2x = 1

3x - 5 = 1

3x = 6

x = 2

Étape 4 : Calculons y en remplaçant x par 2 dans l'expression y = 5 - 2x :

y = 5 - 2(2) = 5 - 4 = 1

Solution : Le système admet une solution unique : (x, y) = (2, 1)

Astuce Cette méthode est particulièrement efficace quand une des inconnues a un coefficient de 1 ou -1 dans une des équations.

2. Méthode par élimination (ou combinaison linéaire)

Cette méthode consiste à :

  1. Multiplier si nécessaire les équations pour que les coefficients d'une des inconnues soient opposés.
  2. Additionner les équations pour éliminer cette inconnue.
  3. Résoudre l'équation à une inconnue obtenue.
  4. Calculer la valeur de la seconde inconnue en la remplaçant dans l'une des équations initiales.

Exemple :

Résoudre le même système :

{
2x + y = 5   (1)
x - y = 1   (2)
}

Étape 1 : Les coefficients de y sont déjà opposés (1 et -1).

Étape 2 : Additionnons les équations (1) et (2) :

(2x + y) + (x - y) = 5 + 1

3x = 6

Étape 3 : Résolvons cette équation :

x = 2

Étape 4 : Calculons y en remplaçant x par 2 dans l'équation (1) :

2(2) + y = 5

4 + y = 5

y = 1

Solution : (x, y) = (2, 1)

3. Méthode de Cramer

Pour un système sous la forme standard :

{
ax + by = c
dx + ey = f
}

On définit les déterminants :

D = \begin{vmatrix} a & b \\ d & e \end{vmatrix} = ae - bd
D_x = \begin{vmatrix} c & b \\ f & e \end{vmatrix} = ce - bf
D_y = \begin{vmatrix} a & c \\ d & f \end{vmatrix} = af - cd

Si D ≠ 0, alors le système admet une solution unique donnée par :

x = \frac{D_x}{D} \quad et \quad y = \frac{D_y}{D}

Exemple :

Résoudre le système suivant avec la méthode de Cramer :

{
2x + y = 5
x - y = 1
}

Ici, a = 2, b = 1, c = 5, d = 1, e = -1, f = 1

Calculons les déterminants :

D = (2)(-1) - (1)(1) = -2 - 1 = -3

D_x = (5)(-1) - (1)(1) = -5 - 1 = -6

D_y = (2)(1) - (1)(5) = 2 - 5 = -3

Donc :

x = D_x / D = -6 / -3 = 2

y = D_y / D = -3 / -3 = 1

Solution : (x, y) = (2, 1)

4. Méthode graphique

Cette méthode consiste à :

  1. Tracer les droites correspondant aux deux équations dans un repère cartésien.
  2. Déterminer les coordonnées du point d'intersection (s'il existe).

Cette méthode est moins précise mais donne une bonne intuition géométrique.

Applications des systèmes d'équations

Problèmes concrets

Les systèmes d'équations à deux inconnues sont utiles pour résoudre de nombreux problèmes concrets :

Exemple : Problème de mélange

Un pharmacien doit préparer 100 ml d'une solution à 30% d'alcool. Il dispose d'une solution à 20% et d'une autre à 50%. Quelles quantités de chaque solution doit-il mélanger ?

Résolution :

Soit x la quantité de solution à 20% et y la quantité de solution à 50%.

Nous avons deux conditions :

  • Volume total : x + y = 100
  • Quantité d'alcool : 0,2x + 0,5y = 0,3 × 100 = 30

On obtient le système :

{
x + y = 100
0,2x + 0,5y = 30
}

Isolons x dans la première équation : x = 100 - y

Substituons dans la seconde : 0,2(100 - y) + 0,5y = 30

20 - 0,2y + 0,5y = 30

20 + 0,3y = 30

0,3y = 10

y = 33,33...

D'où x = 100 - 33,33... = 66,67...

Solution : Le pharmacien doit mélanger 66,7 ml de solution à 20% et 33,3 ml de solution à 50%.

Autres applications

  • Problèmes de prix et de quantités
  • Problèmes de mouvement (distance, vitesse, temps)
  • Calculs financiers (intérêts, investissements)
  • Équilibrage d'équations chimiques
  • Analyse de circuits électriques

Exercices

Exercice 1 : Méthode de substitution

Résoudre le système suivant par la méthode de substitution :

{
3x + 2y = 7
x - y = 2
}

Étape 1 : Isolons x dans la deuxième équation : x = 2 + y

Étape 2 : Substituons dans la première équation : 3(2 + y) + 2y = 7

6 + 3y + 2y = 7

6 + 5y = 7

5y = 1

y = 0,2

Étape 3 : Calculons x : x = 2 + 0,2 = 2,2

Solution : (x, y) = (2,2; 0,2)

Exercice 2 : Méthode d'élimination

Résoudre le système suivant par la méthode d'élimination :

{
4x + 5y = 13
2x - 3y = -5
}

Étape 1 : Multiplions la seconde équation par 2 : 4x - 6y = -10

Étape 2 : Additionnons avec la première équation : (4x + 5y) + (4x - 6y) = 13 + (-10)

8x - y = 3

-y = 3 - 8x

y = 8x - 3

Étape 3 : Substituons dans la première équation : 4x + 5(8x - 3) = 13

4x + 40x - 15 = 13

44x = 28

x = 28/44 = 7/11

Étape 4 : Calculons y : y = 8(7/11) - 3 = 56/11 - 3 = 56/11 - 33/11 = 23/11

Solution : (x, y) = (7/11, 23/11)

Exercice 3 : Méthode de Cramer

Résoudre le système suivant par la méthode de Cramer :

{
5x - 2y = 11
3x + 4y = 1
}

Ici, a = 5, b = -2, c = 11, d = 3, e = 4, f = 1

D = (5)(4) - (-2)(3) = 20 + 6 = 26

D_x = (11)(4) - (-2)(1) = 44 + 2 = 46

D_y = (5)(1) - (11)(3) = 5 - 33 = -28

x = D_x / D = 46 / 26 = 23/13

y = D_y / D = -28 / 26 = -14/13

Solution : (x, y) = (23/13, -14/13)

Exercice 4 : Application concrète

Un cinéma a vendu 200 billets pour une séance, pour un montant total de 1650€. Sachant que le prix d'un billet plein tarif est de 9€ et celui d'un billet réduit est de 6€, combien de billets de chaque type ont été vendus ?

Soit x le nombre de billets plein tarif et y le nombre de billets à tarif réduit.

On a le système :

{
x + y = 200
9x + 6y = 1650
}

De la première équation, on tire : y = 200 - x

En substituant dans la seconde : 9x + 6(200 - x) = 1650

9x + 1200 - 6x = 1650

3x + 1200 = 1650

3x = 450

x = 150

D'où y = 200 - 150 = 50

Solution : Le cinéma a vendu 150 billets plein tarif et 50 billets à tarif réduit.

Cas particuliers

Système sans solution (incompatible)

Si les droites correspondant aux équations sont parallèles, le système n'admet pas de solution.

Exemple :

{
2x + 3y = 6
4x + 6y = 8
}

On remarque que la seconde équation est un multiple de la première (coefficients proportionnels), mais pas le terme constant. Par conséquent, ce système n'admet pas de solution.

Système avec une infinité de solutions

Si les deux équations représentent la même droite, le système admet une infinité de solutions.

Exemple :

{
2x + 3y = 6
4x + 6y = 12
}

La seconde équation est exactement le double de la première. Les solutions sont toutes les paires (x, y) qui vérifient 2x + 3y = 6, ce qui peut s'écrire y = (6 - 2x) / 3 = 2 - 2x/3.

Généralisation à plus de deux inconnues

Les méthodes vues ici peuvent être généralisées pour résoudre des systèmes avec trois inconnues ou plus, bien que la complexité augmente.

Matrices et déterminants

L'étude des systèmes d'équations linéaires est intimement liée à l'algèbre linéaire, notamment aux notions de matrices et de déterminants.

Programmation linéaire

Les systèmes d'équations et d'inéquations linéaires sont à la base de la programmation linéaire, utilisée pour résoudre des problèmes d'optimisation.

Systèmes non linéaires

Il existe également des systèmes d'équations non linéaires (impliquant des produits d'inconnues, des puissances, etc.) qui nécessitent d'autres méthodes de résolution.
