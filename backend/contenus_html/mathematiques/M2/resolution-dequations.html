Résolution d'équations

Méthodes pour résoudre des équations simples

Introduction

La résolution d'équations est une compétence fondamentale en mathématiques qui nous permet de trouver les valeurs inconnues dans une expression algébrique. Que ce soit pour déterminer le prix d'un produit, calculer le temps nécessaire pour accomplir une tâche, ou résoudre des problèmes plus complexes en sciences, savoir résoudre des équations est essentiel.

Dans ce cours, nous allons explorer différentes méthodes pour résoudre des équations simples, en mettant l'accent sur les principes fondamentaux qui vous permettront de développer votre intuition mathématique et d'aborder des problèmes plus complexes avec confiance.

Principes fondamentaux

Avant de plonger dans les méthodes spécifiques, il est important de comprendre les principes de base qui gouvernent la résolution d'équations :

Principe d'équivalence : Lorsque nous effectuons la même opération des deux côtés d'une équation, nous obtenons une nouvelle équation qui a les mêmes solutions que l'équation originale.

En d'autres termes, les opérations suivantes préservent l'égalité et nous permettent de manipuler une équation tout en gardant les mêmes solutions :

  • Ajouter ou soustraire le même nombre des deux côtés
  • Multiplier ou diviser les deux côtés par le même nombre non nul
  • Élever les deux côtés à la même puissance (avec précaution)
  • Prendre la racine carrée des deux côtés (avec précaution)

Notre objectif est toujours d'isoler la variable (généralement notée x) d'un côté de l'équation.

Résolution d'équations du premier degré

Les équations du premier degré (ou équations linéaires) sont de la forme ax + b = c, où a, b et c sont des constantes et a ≠ 0.

Méthode de résolution

  1. Regrouper les termes avec la variable d'un côté de l'équation
  2. Regrouper les termes constants de l'autre côté
  3. Diviser les deux côtés par le coefficient de la variable

Exemple 1 : Résoudre 3x + 5 = 11

3x + 5 = 11
3x = 11 - 5     (on soustrait 5 des deux côtés)
3x = 6
x = 6 ÷ 3     (on divise les deux côtés par 3)
x = 2

Vérification : 3 × 2 + 5 = 6 + 5 = 11 ✓

Exemple 2 : Résoudre 4x - 7 = 2x + 3

4x - 7 = 2x + 3
4x - 2x = 3 + 7     (on regroupe les termes avec x à gauche et les constantes à droite)
2x = 10
x = 10 ÷ 2
x = 5

Vérification : 4 × 5 - 7 = 20 - 7 = 13 et 2 × 5 + 3 = 10 + 3 = 13 ✓

Exercice 1

Résoudre les équations suivantes :

  1. 5x + 3 = 18
  2. 2x - 7 = 3x + 4
  3. 3(x + 2) = 5x - 4

a) 5x + 3 = 18

5x + 3 = 18
5x = 18 - 3 = 15
x = 15 ÷ 5 = 3

b) 2x - 7 = 3x + 4

2x - 7 = 3x + 4
2x - 3x = 4 + 7
-x = 11
x = -11

c) 3(x + 2) = 5x - 4

3(x + 2) = 5x - 4
3x + 6 = 5x - 4
3x - 5x = -4 - 6
-2x = -10
x = 5

Résolution d'équations du second degré

Les équations du second degré sont de la forme ax² + bx + c = 0, où a, b et c sont des constantes et a ≠ 0.

Méthode de résolution par factorisation

Si nous pouvons factoriser le polynôme, nous pouvons utiliser la propriété du produit nul : si A × B = 0, alors soit A = 0, soit B = 0.

Exemple 3 : Résoudre x² - 4x + 3 = 0

x² - 4x + 3 = 0
(x - 3)(x - 1) = 0
x - 3 = 0  ou  x - 1 = 0
x = 3  ou  x = 1

Méthode de résolution par la formule quadratique

Pour toute équation de la forme ax² + bx + c = 0, les solutions sont données par la formule :

x = (-b ± √(b² - 4ac)) / (2a)

Le discriminant Δ = b² - 4ac détermine le nombre de solutions :

  • Si Δ > 0 : deux solutions réelles distinctes
  • Si Δ = 0 : une solution réelle (racine double)
  • Si Δ < 0 : pas de solution réelle (mais deux solutions complexes)

Exemple 4 : Résoudre 2x² + 5x - 3 = 0

a = 2, b = 5, c = -3
Δ = b² - 4ac = 5² - 4 × 2 × (-3) = 25 + 24 = 49
Δ > 0, donc il y a deux solutions réelles distinctes.

x = (-b ± √Δ) / (2a)
x = (-5 ± √49) / (2 × 2)
x = (-5 ± 7) / 4

x₁ = (-5 + 7) / 4 = 2 / 4 = 0.5
x₂ = (-5 - 7) / 4 = -12 / 4 = -3

Équations avec valeur absolue

Les équations contenant des valeurs absolues nécessitent une approche spéciale car |x| représente la distance de x à 0 sur la droite numérique.

Pour résoudre une équation de la forme |expression| = k (où k > 0), nous considérons deux cas :

|expression| = k équivaut à :
expression = k  ou  expression = -k

Exemple 5 : Résoudre |2x - 3| = 7

|2x - 3| = 7

Cas 1 : 2x - 3 = 7
2x = 10
x = 5

Cas 2 : 2x - 3 = -7
2x = -4
x = -2

Les solutions sont x = 5 et x = -2.

Exercice 2

Résoudre les équations suivantes :

  1. |x + 2| = 4
  2. |3x - 1| = 5

a) |x + 2| = 4

Cas 1 : x + 2 = 4
x = 2

Cas 2 : x + 2 = -4
x = -6

Les solutions sont x = 2 et x = -6.

b) |3x - 1| = 5

Cas 1 : 3x - 1 = 5
3x = 6
x = 2

Cas 2 : 3x - 1 = -5
3x = -4
x = -4/3

Les solutions sont x = 2 et x = -4/3.

Équations avec fractions

Pour résoudre des équations contenant des fractions, une approche courante consiste à multiplier tous les termes par le plus petit commun multiple (PPCM) des dénominateurs pour éliminer les fractions.

Exemple 6 : Résoudre (x/2) + (x/3) = 5

(x/2) + (x/3) = 5

Le PPCM de 2 et 3 est 6.
Multiplions tous les termes par 6 :

6 × [(x/2) + (x/3)] = 6 × 5
3x + 2x = 30
5x = 30
x = 6

Vérification : (6/2) + (6/3) = 3 + 2 = 5 ✓

Attention ! Lorsque vous manipulez des équations avec des fractions, assurez-vous que les valeurs de x pour lesquelles un dénominateur s'annule ne sont pas des solutions de l'équation originale.

Exercice 3

Résoudre les équations suivantes :

  1. (1/x) + 1 = 3
  2. (x/4) - (2/3) = (x/6)

a) (1/x) + 1 = 3

(1/x) + 1 = 3
(1/x) = 3 - 1 = 2
1 = 2x
x = 1/2

La solution est x = 1/2. Notez que x ≠ 0 (contrainte due au dénominateur).

b) (x/4) - (2/3) = (x/6)

(x/4) - (2/3) = (x/6)

Le PPCM de 4, 3 et 6 est 12.
Multiplions tous les termes par 12 :

12 × [(x/4) - (2/3) - (x/6)]= 0
3x - 8 - 2x = 0
x - 8 = 0
x = 8

La solution est x = 8.

Applications pratiques

La résolution d'équations est un outil puissant pour résoudre des problèmes de la vie quotidienne. Voici quelques exemples d'applications :

Problèmes de géométrie

Exemple 7 : Calcul de dimensions

Un rectangle a un périmètre de 30 cm. Sa longueur est 2 cm de plus que sa largeur. Quelles sont ses dimensions ?

Soit x = la largeur du rectangle en cm
Alors x + 2 = la longueur du rectangle en cm

Le périmètre est donné par la formule 2 × (longueur + largeur) = 30
2 × [(x + 2) + x] = 30
2 × (2x + 2) = 30
4x + 4 = 30
4x = 26
x = 6.5

Largeur = 6.5 cm
Longueur = 6.5 + 2 = 8.5 cm

Problèmes financiers

Exemple 8 : Calcul d'intérêts

Une somme de 1000€ est placée à un taux d'intérêt annuel. Après un an, elle rapporte 50€ d'intérêts. Quel est le taux d'intérêt ?

Soit r = le taux d'intérêt (en décimal)

Montant des intérêts = Capital × Taux × Temps
50 = 1000 × r × 1
50 = 1000r
r = 50/1000 = 0.05

Le taux d'intérêt est de 0.05, soit 5%.

Exercice 4 : Problème pratique

Un cycliste parcourt 60 km en un certain temps. S'il avait augmenté sa vitesse de 5 km/h, il aurait mis une heure de moins pour faire le même trajet. Quelle était sa vitesse initiale ?

Soit v = la vitesse initiale du cycliste en km/h

Le temps initial est t = distance / vitesse = 60 / v heures

Avec la nouvelle vitesse (v + 5), le temps serait t' = 60 / (v + 5) heures

Selon l'énoncé, t - t' = 1 heure

60/v - 60/(v + 5) = 1
[60(v + 5) - 60v] / [v(v + 5)] = 1
60 × 5 / [v(v + 5)] = 1
300 = v(v + 5)
300 = v² + 5v
v² + 5v - 300 = 0

Résolvons cette équation du second degré :

a = 1, b = 5, c = -300
Δ = b² - 4ac = 5² - 4 × 1 × (-300) = 25 + 1200 = 1225
Δ > 0, donc il y a deux solutions réelles distinctes.

v = (-b ± √Δ) / (2a)
v = (-5 ± √1225) / 2
v = (-5 ± 35) / 2

v₁ = (-5 + 35) / 2 = 15
v₂ = (-5 - 35) / 2 = -20

Comme v représente une vitesse, seule la solution positive est valable. La vitesse initiale du cycliste était donc de 15 km/h.
